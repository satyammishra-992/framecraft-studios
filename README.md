# FrameCraft Studios - Front-End Intern Test (React + Vite + Tailwind)

## Summary
Single-page responsive website inspired by a design reference. Includes Hero, About, Services, Portfolio and Contact sections.
The contact form posts to `https://vernanbackend.ezlab.in/api/contact-us/`.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Postman Collection
`postman_collection.json` included for testing the contact endpoint.

## Notes
- Form validates required fields and email format.
- On successful API response the app displays: "Form Submitted".
